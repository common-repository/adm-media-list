== Description ==

This plugin adds a dashboard widget that gives you the URL for each media library item and allows you to copy the list to your clipboard.

The URL list includes only the original URL and not the URL's for each thumbnail size.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/adm-media-list` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to your dashboard to find the list of Media Library urls.

== Frequently Asked Questions ==

= Does this allow me to download all my media library items too? =

Currently it only provides the URL list, however I am looking to implement this feature in the future.

= Can I get the list of thumbnails too? =

Not at the moment but it's coming!
